# Jetpack Compose

Jetpack Compose is een moderne toolkit voor het bouwen van native Android user interfaces. Compose vereenvoudigt en versnelt de ontwikkeling van gebruikersinterfaces op Android met minder code, 
krachtige tools en intuïtieve Kotlin API's.
Deze opdracht komt uit de Kotlin Course van Google. [Unit 1, Pathway 3](https://developer.android.com/courses/pathways/android-basics-compose-unit-1-pathway-3). 
De reden dat ik met deze course begonnen ben is om meer te leren te krijgen over hoe dat je apps bouwt in Android Studio. Ik wil de basis kennen, 
zodat ik een goed oordeel kan geven over de leercurve van Kotlin in combinatie met Android Studio.

---

## 🚀 Features
- Informatie over Jetpack Compose
- Jetpack Compose UI-Componenten

In deze tutorial heb ik een eenvoudig UI-component met declaratieve functies gebouwd. Ik heb Compose-functies aangeroepen om aan te geven welke elementen er moest worden getoond. 
En de Compose-compiler deed de rest. Compose is gebouwd rondom Composable-functies. 
Met deze functies kun je de UI van je app programmatisch definiëren, omdat je hiermee kunt beschrijven hoe deze eruit moet zien en data-afhankelijkheden kunt opgeven, 
in plaats van je te richten op het proces van de UI-constructie, zoals het initialiseren van een element en het vervolgens koppelen aan een ouderelement. 
Om een ​​Composable-functie te maken, voeg je de annotatie @Composable toe aan de functienaam.
